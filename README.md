# Neurodex

Neurodex is an open-source EMG acquisition and prosthetic control framework using ESP32, BLE communication, and Python-based real-time signal visualization.

## Features

- Real-time EMG signal acquisition
- BLE-based wireless streaming
- Multi-channel EMG support
- Live plotting using Python
- ESP32-C6 firmware support

## Hardware Used

- ESP32-C6 / ESP32-C3 (preferably version 3.0.2)
- BioAmp EXG hardware
- Surface EMG electrodes
- Prosthetic robotic hand

## Future Scope
EMG preprocessing
Gesture classification
Prosthetic hand control
ML-based joint angle estimation
